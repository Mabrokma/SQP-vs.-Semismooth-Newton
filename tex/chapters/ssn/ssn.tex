\chapter{Das halbglatte Newton-Verfahren}
\label{chap:halbglattes_newton_verfahren}

In diesem Kapitel wird das halbglatte\footnote{englisch: semismooth.}
Newton-Verfahren vorgestellt.
Das halbglatte Newton-Verfahren hat eigentlich den gleichen Zweck
wie das Newton-Verfahren,
und zwar ein Gleichungssystem zu lösen.
Man kann aber zeigen,
dass dieses Verfahren zum Lösen
einer restringierten Optimierungsaufgabe
verwendet werden kann.
